Hello! Are you interested in learning how to do things like build web-pages, program secret codes, or write instant messaging programs? If you are join the C. Hunter Ritchie Computer Club!

My name is Jason Mimick, my daughter Bel is in the 3rd grade. I'm a professional software developer and would like to help our children learn more about computers and software.

The club will utilize resources such as coderdojo to provide a real-world software development experience to students. Not only will you learn how to program in languages like JavaScript and Python, you'll also practice working in teams and project organization.

We are in the planning stage for the club, gauging student interest and looking for additional parent voluteers. Please complete this online form http://bit.ly/1w0IQop if you are interested.

Requirements

  • You're in 3rd, 4th, or 5th grade
  • You want to learn how to code
  • You have a laptop you can bring to the club
  • You have email and internet access at home
  • You're able to stay after school one day a week
http://bit.ly/1w0IQop

The club will start shortly after Labor Day - we will be at the Back To School evening event as well. Questions? Please contact Jason - jmimick@gmail.com
